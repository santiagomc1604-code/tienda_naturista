# tienda_naturista
##Exploración de productos saludables: Los clientes pueden navegar por suplementos, hierbas, alimentos orgánicos y productos de bienestar, con filtros por tipo y disponibilidad en stock.

##Carrito de compras dinámico: Permite agregar productos, visualizar cantidades y gestionar compras de manera sencilla.

##Administración de inventario: Panel para el administrador donde se muestran estadísticas de productos, categorías y pedidos, facilitando el control de stock y ventas.

##Gestión completa de productos: Crear, actualizar o eliminar productos con información detallada: nombre, descripción, precio, categoría y cantidad disponible.

##Gestión de categorías naturales: Añadir o eliminar categorías como “Vitaminas”, “Infusiones” o “Alimentos Orgánicos” de manera organizada y sin afectar productos asociados.

##Seguridad y control de acceso: Login para administrador que activa las herramientas de gestión y protege las operaciones críticas de la tienda.
